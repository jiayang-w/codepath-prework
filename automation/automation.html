Project Overview

Automated web searches and scraping processes using Selenium and ChromeDriver to collect square footage information for thousands of industrial addresses from the St. Louis Address & Property Search Website and Regrid.com. Stored the data in CSV/Excel files. Here is a video for my automation project. I put 3 seconds of sleep time at the end of each search to see what's going on. In practice, I can delete those sleep times, and the program can run very fast. The sqft info is not displayed because of the small left-hand side window but the program can scrape it by reading the html code.
